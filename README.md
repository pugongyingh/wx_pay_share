#### js 七种数据类型？

<details><summary><b>答案</b></summary>
<p>

`Boolean`  
`Null`  
`Undefined`  
`Number`  
`String`  
`Object`  
`Symbol` (ECMAScript 6 新定义)  
(ES6 之前)其中 5 种为基本类型:`string`,`number`,`boolean`,`null`,`undefined`  
ES6 出来的 `Symbol` 也是原始数据类型 ，表示独一无二的值  
`Object` 为引用类型(范围挺大),也包括数组、函数

</p>
